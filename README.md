# Calculadora-Cient-fica

https://github.com/priscilo/Calculadora-Cient-fica.git


Calculadora Científica es una app de escritorio moderna creada con React, Vite y Electron. Ofrece una experiencia precisa, accesible y visualmente clara para cálculos científicos avanzados, con una interfaz modular que permite operar de forma intuitiva, rápida y sin distracciones.
